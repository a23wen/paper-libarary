# 🚀 分类导航更新说明

## 新功能：分类快速切换

现在在查看某个分类的论文时，**无需返回上一级**就能快速切换到其他分类！

---

## 📸 功能展示

### 之前的导航流程

```

    ↓ 点击"自然语言处理"
自然语言处理的论文列表
    ↓ 想看计算机视觉？需要返回
    ← 返回所有分类列表
    ↓ 再点击"计算机视觉"
计算机视觉的论文列表
```

**需要 3 次点击** 😓

```

    ↓ 页面顶部显示所有分类标签

    ↓ 直接点击"计算机视觉"
计算机视觉的论文列表
```

**只需 1 次点击** 🎉

## ✨ 功能特点

1. **当前分类高亮**
   - 不同的背景色和字体加粗
   - 一眼就能看出当前在哪个分类

2. **显示论文数量**
   - 每个分类标签后显示 `(N)` 表示论文数
   - 例如：自然语言处理 (3)

3. **响应式设计**
   - 桌面端：横向排列，间距宽松
   - 移动端：自动换行，适应小屏幕

4. **平滑交互**
   - 鼠标悬停时标签会微微上浮
   - 点击后立即跳转到目标分类

## 🎨 视觉设计

### 颜色方案

- **当前分类**：主题色背景 + 白色文字
- **其他分类**：灰色背景 + 正常文字
- **悬停效果**：轻微上浮 + 阴影

### 布局

```css
分类切换栏
├── 图标 + 标题
└── 分类标签（横向排列，自动换行）
    ├── 标签 1 (论文数)
    ├── 标签 2 (论文数) ← 当前高亮
    └── 标签 3 (论文数)
```

## 🎓 最佳实践

1. **首次使用**
   - 点击导航栏的"📂 分类"
   - 选择一个感兴趣的分类进入
   - 顶部会自动显示所有分类标签

2. **日常使用**
   - 不再需要记住如何返回
   - 专注于浏览和切换分类
   - 自然的工作流程

3. **分类规划**
   - 保持分类数量适中（5-15 个）
   - 太多分类会让标签栏过长
   - 定期合并相似分类

## ⚙️ 自定义

### 修改样式

编辑 `assets/css/extended/custom.css`：

```css
/* 修改高亮颜色 */
.category-tag.active {
    background: #your-color;  /* 改成你喜欢的颜色 */
}

/* 修改标签大小 */
.category-tag {
    padding: 0.6rem 1.2rem;  /* 调整内边距 */
    font-size: 1rem;         /* 调整字体大小 */
}
```

### 调整显示位置

编辑 `layouts/categories/list.html`：

```html
<!-- 将标签栏移到底部 -->
<div class="category-switcher" style="margin-top: 3rem;">
  ...
</div>
```

## 🐛 已知限制

1. **首页和归档页面**
   - 此功能仅在分类页面生效
   - 其他页面仍需通过导航栏访问分类

2. **分类数量较多时**
   - 超过 20 个分类可能显得拥挤
   - 建议控制在 15 个以内
